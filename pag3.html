Agenda de Administracion Estudiantil (ADAE)

Esta agenda se realizo en el entre 2020 y 2021.
Es una aplicación de escritorio desarrollada en JAVA y sin conexión a base datos, está aplicacion se realizon con el fin de resolver la problematica Estudiantil en CECyT #9 Juan de Dios Batiz, la aplicacion permitia agregar y modificar unidades academicas en las que se podian agregar y modificar tareas.
La aplicación tambien incluye una sección donde el usuario puede consultar el programas academico de la unidad de aprendizaje


Admintradora de Registros de Mascotas

Está aplicación se realizo en el año 2020, para la empresa "Gatitos Asociation", para el registro y control de las mascotas dentro de un albergue, y consiste de un sistema de registro donde se regitsra la edad, genero, si están vacunado, año de rescate, si están castrados, que animal es e historial medico de cada mascota dentro del albergue, el sistema está programado en JAVA con conexión a base de datos Mysql, y es una aplicacion de escritoyrio

Contestadora de Mensajes Online (C.M.N)

C.M.N. es un servicio online desarrollado en el año 2019, lanzado al mercado en el año 2020, tiene una versión gratis y otra de paga, la aplicacion consiste en que se programa el envio de un mensaje a una cierta hora, el Servicio Web es compatible con redes sociales como Facebook, Instagram, Discord, Telegram, Twitch y TikTok. La versión premiun contiene otras funcionalidades como poder incluir archivos pesados en los mensajes y la opción de programar envios de tareas en plataformas como Classroom, EdModo y Teams
